# Windows Implementation Path

Status: **Reserved**

## Directory roles

- `source/`: future controlled source material.

- `scripts/`: future reviewed Windows automation.
- `documentation/`: Windows-specific procedures and architecture details.

The Windows path shares root governance but requires separate architecture applicability, execution records, validation records, and lifecycle evidence. Linux success never validates Windows behavior.
